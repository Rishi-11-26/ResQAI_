# ResQAI_

AI-Powered Volunteer & Resource Coordination Platform

ResQAI is a data-driven platform designed to help NGOs and community organizations efficiently coordinate volunteers and resources during emergency response and community support operations.

The system centralizes volunteer data, analyzes task requirements, and intelligently matches volunteers to tasks based on skills, location proximity, and availability.

This project was developed as part of **Solution Challenge 2026 India**.

---

## Problem Statement

Community organizations often collect valuable information through field surveys, reports, and volunteer registrations. However, this data is frequently scattered across spreadsheets and documents, making it difficult to identify urgent needs and coordinate volunteers effectively.

As a result:

* Volunteer deployment becomes slow
* Resources are not optimally allocated
* Disaster response coordination becomes inefficient

ResQAI addresses these challenges by providing an intelligent coordination system that helps organizations make faster and better decisions.

---

## Key Features

* Volunteer registration and management
* CSV-based volunteer dataset upload
* Task creation based on community needs
* Intelligent volunteer matching system
* Task prioritization mechanism
* Operational dashboard with analytics
* Volunteer location visualization on map
* Data-driven decision support for coordinators
